# Systeme de gestion d'hotel 
GestionHotel est un système de gestion hôtelière , développé avec PHP, HTML, CSS et MySQL.

L'objectif de ce projet est de gérer les opérations hôtelières — telles que l'affichage des chambres, les réservations et le traitement des paiements — de manière simple et efficace.

Conçu dans un but académique, ce projet met en avant les concepts de développement backend avec PHP et l'intégration de bases de données.
